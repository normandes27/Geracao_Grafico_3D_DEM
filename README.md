 Google Colab, que permite a visualização 3D de Modelos Digitais de Elevação (DEM/MDS). Este recurso, parte da PRAD Toolbox, foi desenvolvido para processar arquivos raster GeoTIFF (.tif) e gerar superfícies 3D com curvas de nível, sendo útil para relatórios e aplicações ambientais, como Projetos de Recuperação de Áreas Degradadas (PRAD). O guia explica o fluxo de trabalho passo a passo (em 7 etapas numeradas), desde a instalação de dependências e o upload do arquivo até o tratamento de dados ausentes e, mais crucialmente, o ajuste de parâmetros editáveis como recorte (CROP), suavização (SMOOTH_SIGMA) e exagero vertical (Z_SCALE). O resultado final do processo é a exportação e download de uma figura 3D em alta resolução no formato PNG.
